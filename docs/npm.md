## Publier un module en 3 lignes

##### En premier, se loguer (ou créer son compte NPM) via le terminal

##### Initialiser le module (cibler le bon dossier avant)
- `npm init`

##### Publier
- `npm publish`

##### Dépublier en cas de boulette
- `npm unpublish ModuleName@1.2.3`
